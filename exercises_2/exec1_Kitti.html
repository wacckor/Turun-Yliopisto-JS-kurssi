Problem description

In this article we discuss the following topics, which we consider very important:
  • Gibberish is poorly understood by most people
  • Gibberish is not considered a valid language
  • The language departments of universities do not teach gibberish
  • Answering exam questions in the gibberish language leads to a failing grade
  • Job applications written in gibberish language are rejected

We argue that gibberish should be officially accepted as a language. We expect there to be counterarguments related to, e.g., gibberish being poorly defined at best. While we acknowledge such concerns...

The way forward

We suggest the following measures to be taken to stregthen the position of gibberish:

  1. More funding for the research of gibberish
  2. Establish support groups for professional gibberish speakers
  3. Increase political pressure to accept gibberish as an official language
  4. Make gibberish the only official language in our country
  5. Agree on punitive measures to those who still use the old languages
We expect these measures to be taken in the order given above. The expected benefits are great, as shown in the table below. The cost-benefit analyses were done using our in-house simulation program called PRNG. The simulation was run repeatedly until we got the results we wanted.
Language Effect on GNP growth Effect on employment rate Effect on IQ
English -0.16 0.01 -9.65
Swedish -0.26 -0.08 1.39
Gibberish 0.81 0.93 8.43

Based on these results, it seems clear that gibberish should be the language of choice in all civilised countries.

Discussion

We showed the benefits of adopting gibberish as the only accepted language. The simulation results, using repeated PRNG, were so clear that we are confident that we should go straight to the implementation phase, instead of conducting human experiments...
